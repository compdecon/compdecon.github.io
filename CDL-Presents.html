CDL Presents ...

Monday March 27th at 8PM, CDL presents:Does Linux Make Sense for You?

Joe Wilkes and Neil Cherry will present a talk to help members decide if they want to install and use the Linux Operating System on their computer. This is important choice since some older computers can't run the newer Windows version but still may work fine. Linux could be an option for the average user.

This talk is for beginners with Linux. It will show what applications are available for Linux to meet the needs of basic users of a computer. It will not show how to install Linux (future class).

The class is free to all members, membership is $30 per month or $300 per year.
